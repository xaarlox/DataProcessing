# Job Scraper (Work.ua & DOU.ua)

Проєкт є інтрументом для автоматизованого збору даних про вакансії з двох популярних українських ресурсів: Work.ua та DOU.ua.

## Архітектура

Програма побудована на принципах ООП:

+ **Абстрактний клас JobScraper** служить шаблоном. Містить спільну для всіх сайтів логіку: завантаження сторінки з мережі та створення фінальної таблиці.
+ **Класи DouScraper та WorkUaScraper** спеціалізовані "робітники". Кожен із них знає унікальну HTML-структуру свого сайту і вміє витягувати з неї назву вакансії, компанію та зарплату.

## Алгоритм роботи
1. *Запит*: Програма формує URL-адресу з пошуковим словом.
2. *Завантаження*: Виконується HTTP-запит до сайту. Якщо сайт доступний, отримується його HTML-код.
3. *Парсинг*: BeautifulSoup шукає конкретні блоки на сторінці, наприклад, теги `<li>` на DOU або `<div>` на Work.ua, і витягує з них текст.
4. *Обробка*: Текст чиститься від зайвих пробілів та технічних символів (`\xa0`, `\u202f`).
5. *Об'єднання*: Дані з усіх сайтів збираються в один pandas.DataFrame, де додається колонка Source.
